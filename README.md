# banco_de_dados_DIO
Desafio 3: Potência Tech powered by iFood | Ciência de Dados - Criando bancos de dados usando mySQL

DESCRIÇÃO DO DESAFIO 1: Neste desafio, você terá a oportunidade de criar seu primeiro projeto lógico de banco de dados utilizando o MySQL. O objetivo é replicar a modelagem de um banco de dados para um cenário de e-commerce. Prepare-se para aplicar seus conhecimentos em modelagem de banco de dados e traduzir os requisitos do cenário em uma estrutura lógica coerente usando o MySQL (Banco da dados de e-commerce).

DESCRIÇÃO DO DESAFIO 2: Neste desafio, você irá utilizar seu esquema conceitual, criado no desafio do módulo de modelagem de BD com modelo ER, para criar o esquema lógico para o contexto de uma oficina. Você definirá todas as etapas. Desde o esquema até a implementação do banco de dados (Banco de dados de oficina mecânica).

Abaixo seguem algumas das perguntas que foram possíveis exclarecer e dados que foram possíveis demonstrar:

-- Visualização dos dados
-- Andamento dos pedidos do clientes
-- Descrição completa dos pedidos dos clientes ordenado por urgencia
-- Numero de pedidos realizados pelos clientes
-- Clientes com problemas recorrentes
-- Descrição completa dos pedidos dos clientes
-- Relação entre produtos, fornecedores e estoque
-- Informações completas sobre os vendedores
